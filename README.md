## 📌 Sobre o Projeto
O **Bar Manager** é um sistema web robusto desenvolvido para otimizar a operação de bares e pequenos comércios. O foco principal é a agilidade no atendimento e a precisão no controle de ativos, permitindo uma visão clara da saúde financeira do negócio.

O projeto soluciona a complexidade de gerenciar múltiplos itens de estoque e fluxos de caixa variáveis em tempo real.

---

## ✨ Funcionalidades Principais
*   **📦 Gestão de Produtos:** Cadastro completo de itens com suporte a upload de fotos para identificação visual rápida.
*   **📉 Controle de Estoque:** Atualização em tempo real após cada venda, evitando rupturas ou perdas.
*   **💰 Fluxo de Caixa:** Monitoramento de entradas e saídas diárias de forma simplificada.
*   **📋 Histórico de Vendas:** Registro detalhado de transações para acompanhamento de performance e fechamento de conta.
*   **🚀 Eficiência Operacional:** Interface intuitiva desenvolvida para reduzir o tempo de resposta entre o pedido e a baixa no sistema.

---

## 🛠️ Tecnologias Utilizadas
*   **Back-end:** PHP (Lógica de vendas e manipulação de arquivos/fotos).
*   **Banco de Dados:** MySQL (Persistência de dados de produtos, estoque e vendas).
*   **Front-end:** HTML5, CSS3 e JavaScript (Interface responsiva e dinâmica).
*   **Ambiente de Dev:** Laragon (WAMP Stack).

---

## 📂 Estrutura Sugerida
*   **`produtos/`**: Módulo de cadastro e exibição de itens.
*   **`vendas/`**: Lógica de checkout e baixa no estoque.
*   **`img/`**: Pasta destinada às fotos dos produtos cadastrados.
*   **`caixa/`**: Gestão financeira e relatórios de fluxo.
